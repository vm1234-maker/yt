## Quality requirements
- **Format**: MP3 (320kbps) or WAV/FLAC preferred
- **Duration**: 5 minutes minimum (longer = less noticeable loop point)
- **Looping**: The file should loop seamlessly — the end should flow naturally into the beginning
- **Sample rate**: 44.1kHz or 48kHz
- **License**: CC0 only (no attribution, no restrictions, safe for monetized YouTube)

## Finding CC0 files on Freesound
1. Go to freesound.org
2. Search for your term
3. In filters, set **License** → **Creative Commons 0**
4. Sort by **Downloads** (most downloaded = most vetted quality)
5. Download the highest quality version (WAV if available)
6. Rename to the filename listed above

## Tips for seamless loops
- Look for files explicitly tagged "loop" or "seamless"
- Preview to check the loop point — it should not have a sudden cut
- Files from professional sound designers (high download counts) are usually pre-looped
- FFmpeg will loop any file infinitely regardless, but a file with a clean loop point sounds far better
